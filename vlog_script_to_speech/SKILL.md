# Vlog Script to Speech Skill

本 Skill 用于将用户的手写稿转换为适合朗读的口播稿。

## 输入

用户需要提供：
- 手写稿内容（可以是 Markdown 或纯文本格式）
- 节目风格偏好（可选：轻松活泼、专业严肃、亲切自然等）

## 输出

生成口播稿，包含：
- 开场白
- 正文内容（适合朗读的句子）
- 过渡语
- 结尾语

## 口播稿特点

1. **口语化**：
   - 使用自然的表达方式
   - 避免书面语的生硬感
   - 加入适当的语气词

2. **节奏感**：
   - 句子长度适中
   - 有明确的停顿点
   - 适合配音/朗读

3. **互动性**：
   - 适当的提问或互动
   - 引导观众思考
   - 增加观众参与感

## 执行步骤

1. 询问用户手写稿内容或让用户提供文件路径
2. 分析手写稿的核心内容
3. 将书面语转换为口语化表达
4. 添加开场白、过渡语、结尾语
5. 输出完整的口播稿

## 输出格式

```markdown
# [节目标题]

## 开场白
（用于吸引观众注意力的开场）

## 正文
（按段落/场景组织的内容）

## 结尾语
（感谢观众、引导互动）
```

完成后告知用户：
- 生成的口播稿内容
- 朗读建议（重点强调、情感处理等）
- 预计朗读时长
