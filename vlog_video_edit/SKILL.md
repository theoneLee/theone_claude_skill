# Vlog Video Edit Skill

本 Skill 用于对用户录制的 vlog 视频进行精细剪辑。

## 输入

用户需要提供：
- 原始视频文件路径（支持 MP4、MOV 等常见格式）
- 口播稿（用于参考内容，方便定位需要剪辑的位置）
- 任何特殊的剪辑要求

## 剪辑原则

1. **去除内容**：
   - 重复的语句
   - 说错/卡壳的部分
   - 长时间的停顿
   - 无意义的口头禅（过于频繁的）

2. **保持通顺**：
   - 保留必要的语气词（嗯、啊、这个等）
   - 保留自然的过渡语
   - 保持语句的连贯性
   - 不让观众感到剪辑感太强

3. **节奏把握**：
   - 不要剪得太紧凑
   - 保留适当的口语节奏
   - 整体流畅度优先

## 执行步骤

1. 询问用户视频文件路径和口播稿
2. 加载视频文件，分析内容
3. 识别需要剪辑的位置：
   - 让用户观看视频并标记需要剪辑的位置
   - 或者让用户提供口播稿中对应的位置描述
4. 执行剪辑操作
5. 导出剪辑后的视频
6. 让用户预览，确认效果

## 剪辑技巧

- **过渡自然**：使用淡入淡出或画面切换来掩盖剪辑点
- **语气词保留**：适当的"嗯""啊"让视频更自然
- **句子完整**：尽量保持句子的完整性，避免截断
- **情绪连贯**：注意前后的情绪和语速匹配

## 输出

完成后告知用户：
- 剪辑后的视频文件路径
- 剪辑了哪些部分（简要说明）
- 建议的预览重点
