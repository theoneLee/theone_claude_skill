# Vlog Subtitle Skill

本 Skill 用于为视频自动生成并配置字幕。

## 输入

用户需要提供：
- 视频文件路径（已剪辑完成的视频）
- 字幕样式偏好（可选）

## 字幕配置选项

1. **样式选择**：
   - 字体（默认使用清晰易读的无衬线字体）
   - 大小（中号，适合移动端观看）
   - 颜色（白色带黑色描边，确保清晰）
   - 位置（底部居中）

2. **格式选择**：
   - SRT 字幕文件
   - ASS 高级字幕（支持更多样式）
   - 内嵌字幕（烧录到视频中）

## 执行步骤

1. 询问用户视频文件路径
2. 使用语音识别技术生成字幕：
   - 可以使用 whisper 等工具进行语音识别
   - 或者让用户提供口播稿作为字幕文本
3. 校对字幕（如果有口播稿对照，检查是否匹配）
4. 配置字幕样式
5. 输出最终带字幕的视频或字幕文件

## 字幕校对要点

- 检查时间轴是否准确
- 确认文字内容正确
- 调整句子分割使其自然
- 保持适当的每行字数

## 输出

完成后告知用户：
- 带字幕的视频文件路径（或字幕文件路径）
- 字幕格式
- 如需调整可以修改的内容
