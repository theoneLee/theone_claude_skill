# 口播稿转换 Prompt 模板

请将以下手写稿转换为适合口播的演讲稿。

## 要求

1. **口语化**：将书面表达转为自然的口头表达，像在和朋友聊天一样
2. **节奏感**：句子长度控制在 10-25 字之间，给朗读留出自然停顿
3. **互动性**：适当加入与观众的互动，如提问、引导思考
4. **过渡自然**：章节之间使用自然的过渡语连接
5. **PPT 对应**：标注每段对应的 PPT 页码（用 `<!-- 对应 PPT 第 X 页 -->` 标记）

## 结构要求

- 必须包含：开场白、正文（按章节）、结尾语
- 开场白要吸引注意力，可以用一个问题或有趣的故事开头
- 结尾语要引导观众互动（关注、点赞、评论）
- 每个章节准确对应 PPT 页码

## 风格

- 亲切自然，不要太正式
- 适当使用"其实"、"那么"、"你看"、"对吧"等语气词
- 重要观点用简短有力的句子强调
- 数据/技术术语用通俗易懂的方式解释

## 手写稿内容

```
{draft_content}
```

请生成完整的口播稿，按照上述结构和要求输出。在开头标注预计朗读时长（按每分钟 200 字估算）。
