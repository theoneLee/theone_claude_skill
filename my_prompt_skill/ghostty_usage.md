## 分屏
cmd+d 水平分屏（左右）
cmd+shift+d 垂直分屏

Cmd + Option + 方向键 (分屏后切换焦点) 

cmd+w 关闭标签页/分屏
Cmd + Shift + T (undo。重新打开刚才关闭的那个窗口)  

## 默认快捷键
ghostty +list-keybinds --default 输出的是它内置的所有默认快捷键

## 其他快捷方式
Cmd + Shift + P 弹出一个搜索框,搜索命令

Cmd + Shift + J (write_screen_file) 将当前终端屏幕上看到的所有文本导出为一个文件。

super + ctrl  + f               toggle_fullscreen 切换到全屏
