# Vlog Script to PPT Skill

本 Skill 用于将用户的手写稿转换为视觉吸引力强的 PPT。

## 输入

用户需要提供：
- 手写稿内容（可以是 Markdown 或纯文本格式）
- PPT 主题/标题
- 期望的页数（可选，默认 10-15 页）

## 输出

生成一个 PPT 文件，包含：
- 封面页（标题、副标题）
- 内容页（图文并茂）
- 过渡页
- 结尾页

## 设计原则

1. **视觉吸引力**：
   - 使用鲜明的配色方案
   - 适当的图标和插图
   - 清晰的层次结构

2. **内容组织**：
   - 每页一个核心观点
   - 文字简洁有力
   - 配合适当的图片/图表

3. **适合口播**：
   - 内容节奏适合朗读
   - 页面停留时间合理

## 执行步骤

1. 询问用户手写稿内容或让用户提供文件路径
2. 分析手写稿结构，提取关键要点
3. 设计 PPT 架构和每页内容
4. 生成 PPT 文件（使用 Python pptx 库或用户指定工具）
5. 告知用户生成的 PPT 文件路径

## 输出格式

完成后告知用户：
- 生成的 PPT 文件路径
- 建议的演示时间
- 如有需要可以调整的内容
